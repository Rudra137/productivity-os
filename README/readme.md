# 🚀 Personal Productivity OS

A centralized digital control center designed to manage tasks, habits, and notes in one place. Built to bridge the gap between planning and execution.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Transitioning to React)
- **Version Control:** Git & GitHub
- **Status:** In-Development (MVP Stage)

## 🌟 Key Features (Planned)
- **Smart Task Dashboard:** Prioritize tasks using the Eisenhower Matrix.
- **Habit Tracker:** Visual streaks to maintain consistency.
- **Quick Notes:** A "scratchpad" for instant ideas.

## 📂 Project Structure
- `index.html`: The core structure of the dashboard.
- `styleindex.css`: Custom UI/UX styling.

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com`
2. Open `index.html` in your browser.
